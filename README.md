# Microsoft Azure Database Interface 
This project is a Java/database application built to interface with a Microsoft Azure database. It follows the teacher’s template while adding extra UI elements and features to enhance functionality.

### Key Features
* Database Connectivity: Seamless connection to Microsoft Azure.

* UI Enhancements:

Custom graphical interface with dynamic elements.

Splash screen with fade transition.

Login screen with test credentials (e.g., example@gmail.com).

* Additional UI Functions:

Light & Dark theme toggling.

Profile picture update.

About page accessible through the menu.

* Keyboard Shortcuts:

Ctrl + A, Ctrl + E, Ctrl + C, and Ctrl + D for quick actions.

## Main Components
* DB_GUI_Controller: Manages the user interface and database interactions.

* AboutPageController: Controls the About page details.

* DB_Application: The main entry point of the application.

* SplashScreenController: Handles the splash screen display.

* LoginScreenController: Manages user login.

* Person: Represents a user with associated details.

## How to Run
1) Clone the Repository:

git clone https://github.com/your-username/azure-database-interface.git
cd azure-database-interface

2) Open the project in your preferred Java IDE.

3) Run DB_Application to start the application.
